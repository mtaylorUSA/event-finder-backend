
PocketBase Fields/choices w/in Collections (general) Generalized 

1. To Add a New Collection to PocketBase
* Click “+ New collection” button
* Add name
* Specify Type 
o Base Collection
o View Collection
o Auth Collection
* Use the “Fields” Tab (not the “API Rules” tab) 
* Specify the kind of field. Choices are:
o T = Plain Text
o pencil icon = Rich editor
o # icon = Number
o toggle icon = Bool
o envelope icon = Email
o chain link icon = Url
o calendar icon = DateTime
o bullet list icon = Select
o picture icon = File
o flow chart connector icon = Relation
o {} = JSON
* Click the “Create” button

2. To Add Settings to Collections
* Click the named “Collection” box
* Click the Sprocket icon

* Field type: T = Plain Text	

o name: (text box)
o click sprocket icon
o min length: (text box)
o max length: (text box)
o Regex pattern: (text box)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)
* Field type: pencil icon = Rich editor
o name: (text box)
o Click sprocket icon
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)
o strip urls domain: (toggle on/off)

* Field type: # icon = Number
o name: (text box)
o then click sprocket icon
o Min: (text box)
o Max: (text box)
o Nonzero: (toggle on/off)
o Presentable: (toggle on/off)
o No decimals: (toggle on/off)

* Field type: toggle icon = Bool
o name: (text box)
o Click sprocket icon
o Nonfalsey: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: envelope icon = Email
o name: (text box)
o Click sprocket icon
o Except domains: (text box)
o Only domains: (text box)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: chain link icon = Url
o name: (text box)
o Click sprocket icon
o Except domains: (text box)
o Only domains:  (text box)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: calendar icon = DateTime
o name: (text box)
o Click sprocket icon
o min date (UTC):  (text box)
o max date (UTC):  (text box)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: bullet list icon = Select
o name: (text box)
o Click sprocket icon
o Choices: (text box) - items Must be entered in a single line with the items separated by commas.
o Single/multiple: (picklist)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: picture icon = File
o name: (text box)
o Click sprocket icon
o Single/multiple: (picklist)
o Allowed types: (picklist)
o Choose presets: (picklist)
o thumb sizes: (text box)
o Max file size: (text box)

SizeBytes1 MB10485765 MB524288010 MB1048576025 MB2621440050 MB52428800
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)
o Protected: (toggle on/off)

* Field type: flow chart connector icon = Relation
o name: (text box)
o Click sprocket icon
o Select Collection: (picklist)
o Single/multiple: (picklist)
o Cascade delete: (picklist true/false)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)

* Field type: {} = JSON
o name: (text box)
o Click sprocket icon
o Max size (bytes): (text box)
o String value normalizations: (picklist)
o Nonempty: (toggle on/off)
o Presentable: (toggle on/off)
